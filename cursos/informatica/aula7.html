Aula 7: Compactadores (WinZip/WinRAR)

O que é o WinZip?

O WinZip é um aplicativo que compacta arquivos, reduzindo seu tamanho por meio de algoritmos de compressão. É útil para envio por e-mail ou armazenamento em dispositivos como pendrives.

Arquivos compactados possuem extensões como: .zip, .rar, .arj, .tar, .lzh, entre outros.

Como compactar arquivos

  1. Selecione os arquivos desejados.
  2. Clique com o botão direito do mouse e escolha a opção WinZip.
  3. Escolha entre:
    • Add to Zip file: permite escolher nome e local do arquivo compactado.
    • Add to [nome].zip: cria o arquivo automaticamente com nome sugerido.
  4. Defina o modo de compactação:
    • Normal
    • Maximum (enhanced deflate)
    • Super fast
  5. Se necessário, habilite a opção de múltiplos volumes para dividir o arquivo em partes menores.

Como descompactar arquivos

  1. Dê dois cliques no arquivo compactado.
  2. O WinZip abrirá e mostrará os arquivos contidos.
  3. Clique em Extract e escolha o local onde os arquivos serão salvos.
  4. Confirme e aguarde o processo de extração.

WinSCP

O WinSCP é um aplicativo para transferência de arquivos entre computadores locais e servidores remotos via protocolo SSH.

  1. Execute o aplicativo.
  2. Informe:
    • Host name: nome do servidor
    • Port number: geralmente 22
    • User name: nome do usuário
    • Password: senha do usuário
  3. Clique em Login para conectar.
  4. Arraste os arquivos entre as janelas local e remota para transferir.
  5. Ao finalizar, desconecte a sessão para segurança.

Scanner

Equipamento que digitaliza imagens e textos, convertendo-os em formato digital para edição e armazenamento.

  1. Coloque a imagem na plataforma e feche a tampa.
  2. Execute o software do scanner.
  3. Faça uma pré-visualização (Preview).
  4. Selecione a área desejada.
  5. Configure os parâmetros: tipo de imagem, resolução, brilho, contraste, formato de arquivo.
  6. Execute a digitalização.
  7. Salve o arquivo gerado.

Edição de Imagens

Aplicativos permitem melhorar imagens digitalizadas com recursos como:

  • Brilho / Contraste
  • Recortar / Copiar / Inserir
  • Negativo / Espelho
  • Colorir / Preto e branco
  • Efeitos especiais: neblina, vidro, rascunho, etc.

Recomenda-se sempre salvar uma cópia da imagem original antes de editar.
