Aula 9: Estrutura do Computador

Unidade de disco (lógica)

É a identificação usada para cada unidade de disco instalada no computador, como HDs e leitores de CD/DVD.

  • C: geralmente é o disco rígido principal.
  • D: pode ser uma partição do mesmo HD ou outro disco.
  • E: normalmente é o leitor de CD/DVD.

As letras podem variar conforme a configuração do sistema.

Diretório ou pasta

São divisões dentro dos discos para organizar arquivos. O usuário pode criar, renomear e excluir pastas.

Algumas pastas são criadas automaticamente pelo sistema ou por programas instalados e não devem ser apagadas sem conhecimento.

Arquivo

É a forma como as informações são armazenadas no computador. Cada arquivo possui:

  • Nome: identificação do conteúdo (ex: alunos).
  • Tipo: extensão que indica o programa associado (ex: .doc, .jpg, .pdf).

Rede de computadores

É a conexão entre dois ou mais computadores para compartilhar recursos e informações.

Vantagens de uma rede:

  • Compartilhar impressoras, arquivos e internet.
  • Facilidade na troca de dados entre máquinas.

O compartilhamento pode ser configurado para permitir ou restringir o acesso a determinados recursos.
